Scrapping

Python

Extraction et analyse de données web

Le web scraping est une technique d’extraction automatisée de données à partir de sites web.

Recherche de véhicules d’occasion (web scraping d’un site d’annonces)

Dans ce second projet, plus complexe, j’ai développé un outil d’extraction de données à partir d’un site d’annonces automobiles. L’objectif était de récupérer les annonces correspondant à des critères de recherche spécifiques (marque, modèle, kilométrage, prix, etc.). Comme pour le projet précédent, les données étaient stockées dans une base de données SQLite afin de pouvoir les analyser et suivre les tendances du marché. Ce projet m’a permis d’approfondir la gestion de scraping à grande échelle et l’analyse de données structurées.

Suivi des prix de cartes graphiques (web scraping d’un site e-commerce)

Ce projet avait pour objectif de suivre l’évolution des prix des cartes graphiques sur un site de vente en ligne. Il s’agissait de mon premier projet de web scraping. J’ai utilisé Python avec les bibliothèques requests et BeautifulSoup pour extraire les données. Les prix récupérés étaient ensuite stockés quotidiennement dans une base de données SQLite grâce à un script automatisé (via cron ou schedule en Python). Ce projet m’a permis de constituer un historique de l’évolution des prix dans le but d’en analyser la variance au fil du temps.
