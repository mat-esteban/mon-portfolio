Cocci

HTML | CSS | JavaScript

Création d’une page vitrine pour un commerce local (HTML/CSS/JavaScript)

Présentation

Dans le cadre d’un projet personnel, je développe actuellement une page web statique pour le magasin CocciMarket situé à Plonévez-Porzay

  • Learn more

Objectifs

. L’objectif est de proposer une vitrine simple et efficace présentant :

• Les horaires d’ouverture du magasin ;

• Une sélection de produits disponibles en rayon.

  • Learn more

Compétences

J’ai utilisé HTML et CSS pour la structure et le design, et intégré JavaScript afin d’ajouter des fonctionnalités interactives comme le filtrage de produits, l’actualisation automatique de certaines données, ou encore une interface plus fluide pour la navigation.

Ce projet me permet de consolider mes compétences en développement front-end tout en répondant à un besoin concret d’un commerce local.
