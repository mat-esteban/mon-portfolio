Outil d'analyses

Python | Pandas | Matplotlib

Présentation

J’ai conçu un outil d’analyse de performances pour joueurs de football, développé en Python. Ce script permet de suivre les statistiques individuelles d’un joueur au cours d’un match à partir de données brutes (fichiers CSV ou API sportives).

Objectifs

L’outil génère automatiquement :

• Des indicateurs clés (passes réussies, tirs, duels gagnés, etc.) ;

• Des visualisations de performance sous forme de graphiques (radar chart, heatmap, barres) ;

• Un résumé statistique exportable.

Problématique

Ce projet est né de ma passion pour le football et m’a permis de combiner analyse de données et visualisation de manière utile pour un usage amateur ou semi-professionnel.
