Générateur de mots de passe personnalisés

Python

Problématique

Utilisant régulièrement un gestionnaire de mots de passe (KeePass), j’ai constaté qu’il manquait une fonctionnalité : la personnalisation avancée de la génération de mots de passe.

Objectif

J’ai donc développé mon propre script Python permettant de créer des mots de passe sécurisés selon des critères précis définis par l’utilisateur :

• Nombre de caractères ;

• Inclusion ou non de lettres majuscules, minuscules, chiffres, caractères spéciaux ;

• Options d’exclusion (caractères ambigus, répétitifs, etc.).

Indicateur de robustesse

Un indicateur intégré évalue automatiquement la robustesse du mot de passe généré (faible, moyen, fort) sur la base d’algorithmes simples d’entropie. Ce projet m’a permis de travailler la gestion des entrées utilisateurs, la sécurité, et l’expérience utilisateur (UX) en ligne de commande.
