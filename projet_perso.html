Projets de SD1

Application Excel automatisée

Ce projet consistait à créer une application automatisée sur Excel pour aider les étudiants du BUT1 SD à calculer leurs moyennes générales et par compétence. L'automatisation a été réalisée principalement grâce au code VBA.

Régression sur des données réelles

Ce projet visait à concevoir un modèle statistique de prédiction de la valeur foncière sur 2000 biens immobiliers vendus en 2021 dans les départements des Deux-Sèvres et de Charente-Maritime.

Transformation JSON en CSV

Ce projet impliquait la transformation d'un fichier JSON en format CSV à l'aide de Python, en tenant compte de la structure des données et des besoins du donneur d'ordre.

Implémentation d’une base de données

L'objectif était d'implémenter des données dans une base de données MySQL en utilisant des requêtes SQL et du code Python pour lire et insérer les données à partir d'un fichier CSV.
